# Millennium Theme Submission Draft

Issue target: https://github.com/SteamClientHomebrew/Millennium/issues

## Title

Theme submission: UMI

## Body

### Theme name

UMI

### Repository

https://github.com/alexh/umi-millennium-theme

### Description

UMI is a vintage dark industrial Steam theme with graphite chrome, cold light text, black inset fields, orange-edged steel bevels, scanline grain, and hazard-orange activation states.

### Tags

Dark, Vintage, Industrial, Textured, Ornate

### Notes

The theme uses explicit Millennium `Patches` entries with `MatchRegexString` and `TargetCss`/`TargetJs`. `webkit.css` is injected globally, with per-window CSS for Library, Friends, and Big Picture.
